# Military History Map

A simple webmap that displays military engagements throughout history and filterable by time. 

**[Live Demo](https://your-vercel-url.vercel.app)**

# Built With

- [MapLibre GL JS](https://maplibre.org/) — vector tile map rendering
- [React](https://react.dev/) — UI framework
- [Vite](https://vitejs.dev/) — build tool
- [MapTiler](https://www.maptiler.com/) — map tile provider

# Data

Conflict data sourced from [Wikidata](https://www.wikidata.org/)

The `scripts/` folder contains a Python script that creates a sample battle dataset.
